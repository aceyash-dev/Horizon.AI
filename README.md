# **​🌌 Horizon.AI**
My Universal AI Web Agent, **Horizon.AI** is a high-performance, free web-based chatbot UI that bridges the gap between you and the world’s most powerful Large Language Models. No subscriptions, no middleman—just your API key and limitless intelligence.
## **​✨ Key Features**
> ​Multi-LLM Support: Toggle instantly between models from OpenRouter (Claude, GPT, Llama) and Google AI Studio (Gemini 3 Pro/Flash).

> ​Horizon UI: A premium & modern interface getting updated as per feedback. 

__April Beta__

 New Features & Hotfixes:

•​Split Top Bar: Dedicated space for navigation and model status.   

•​Revamped Sidebar: Includes a real-time Model Stats card for token tracking.  

•​Pill-Shaped Input: Modern, ergonomic chat entry field.  

> ​Client-Side Privacy: Your API keys are stored locally. We don't see your data; you control the connection.
​Firebase Integration: Secure authentication and persistent session history so you never lose a prompt.

### 🔑 API Configuration

1.  **Google AI Studio:** Visit [<u>**Google AI Studio**</u>](https://aistudio.google.com/) to generate your Gemini API key.
2.  **OpenRouter:** Sign up at [<u>**OpenRouter**</u>](https://openrouter.ai/) to access a unified interface for 300+ models.
