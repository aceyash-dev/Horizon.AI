# 🌟 Why Horizon.AI?

Unlike many AI clients, Horizon.AI focuses on speed, elegance, and openness.

- Beautiful interface
- Bring your own API key
- Multiple providers
- Works entirely in the browser
- Lightweight
- Mobile-first
- Easy to modify
- No backend required (except authentication)
